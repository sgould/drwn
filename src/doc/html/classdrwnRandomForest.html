Detailed Description

Implements a Random forest ensemble of decision trees classifier. See L. Breiman, "Random Forests", Machine Learning, 2001.

The following code snippet shows example learning a random forest classifier on a training dataset and then testing it on a hold out evaluation dataset.

// load training dataset
drwnClassifierDataset dataset;
dataset.read("training_data.bin");
// train the classifier
const int nFeatures = dataset.numFeatures();
const int nClasses = dataset.maxTarget() + 1;
drwnRandomForest model(nFeatures, nClasses);
model.train(dataset);
// load evaluation set
dataset.read("testing_data.bin", false);
// predict labels
vector<int> predictions;
model.getClassifications(dataset.features, predictions);
