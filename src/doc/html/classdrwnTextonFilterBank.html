Detailed Description

Implements a 17-dimensional filter bank.

The filter bank is defined in:

  • J. Shotton, J. Winn, C. Rother, A. Criminisi. "TextonBoost for Image Understanding: Multi-Class Object Recognition and Segmentation by Jointly Modeling Texture, Layout, and Context," IJCV 2008.
  • J. Winn, A. Criminisi, and T. Minka. "Categorization by learned universal visual dictionary," ICCV 2005.

The features produced by this filterbank are used in multi-class image segmentation (see Multi-class Image Segmentation).
